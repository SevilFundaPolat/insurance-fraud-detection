# 🛡️ Insurance Fraud Detection using Explainable Machine Learning

## Project Overview

Insurance fraud is one of the biggest challenges faced by insurance companies, resulting in billions of dollars in financial losses every year.

This project aims to build an end-to-end Machine Learning solution capable of detecting potentially fraudulent insurance claims while providing transparent and interpretable predictions.

---

## Business Problem

Insurance companies receive thousands of claims every day.

Investigating every claim manually is expensive and time-consuming.

The objective is to develop a predictive model that identifies suspicious claims and helps investigators prioritize high-risk cases.

---

## Project Objectives

- Understand the business problem
- Explore and analyze insurance claims data
- Clean and preprocess the dataset
- Engineer meaningful features
- Train and compare Machine Learning models
- Explain predictions using SHAP
- Deploy the model with Streamlit
